# Blockchain Glossar

## Wallet
Eine Wallet ist ein digitales Konto zur Verwaltung von Kryptowährungen und Token. Sie enthält einen öffentlichen Schlüssel (Adresse) und einen privaten Schlüssel (Passwort für Signaturen). Tokens liegen nicht in der Wallet selbst, sondern im Smart Contract. Die Wallet kontrolliert lediglich den Zugriff.

## Öffentliche Adresse
Die öffentliche Adresse ist vergleichbar mit einer IBAN. Jeder darf sie kennen, da sie nur zum Empfangen von Kryptowährungen verwendet wird.

## Privater Schlüssel
Der private Schlüssel ist geheim. Er ermöglicht das Signieren von Transaktionen. Wer diesen Schlüssel besitzt, kann im Namen der Wallet handeln.

## Gas
Gas ist die Gebühr, die bei jeder Ethereum-Transaktion anfällt. Es bezahlt die Rechenarbeit der Validatoren. Gas wird in Gwei gemessen, ein Bruchteil von Ether.

## Gas Price und Gas Limit
- **Gas Price**: Preis pro Einheit Gas  
- **Gas Limit**: Maximale Menge Gas, die eine Transaktion verbrauchen darf

## Smart Contract
Ein Programm auf der Blockchain, das automatisch ausgeführt wird. Einmal deployed kann es nicht mehr verändert werden.

## ERC-20 Token
Ein Standard für fungible Token auf Ethereum. Jeder ERC-20 Token besitzt:
- Name  
- Symbol  
- Dezimalstellen  
- Funktionen wie `transfer()`, `balanceOf()`

## Transaction Hash
Eine eindeutige Kennung, die eine Transaktion im Blockchain-Netzwerk identifiziert.
