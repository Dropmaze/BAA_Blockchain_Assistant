## Projektbeschreibung
Dieses Projekt entwickelt einen Blockchain Assistant, der es ermöglicht, über natürliche Sprache mit Smart Contracts zu interagieren.  
Ziel ist die Verbesserung der Benutzerfreundlichkeit von Blockchain-Technologien durch natürliche Sprachsteuerung und erklärbare Interaktionen.
